INHERITANCE

Answer: Inheritance is a mechanism where one class (subclass) inherits properties and behaviors from another class (superclass) using the extends keyword.

  • Single (supported)
  • Multilevel (supported)
  • Hierarchical (supported)
  • Multiple inheritance using classes is not supported (but can be achieved using interfaces)
class Animal {
  void sound() {
    System.out.println("Animal makes sound");
  }
}
class Dog extends Animal {
  void bark() {
    System.out.println("Dog barks");
  }
}

Answer: When a subclass provides a specific implementation of a method already defined in its superclass.

class Animal {
  void sound() {
    System.out.println("Generic sound");
  }
}
class Cat extends Animal {
  @Override
  void sound() {
    System.out.println("Meow");
  }
}
