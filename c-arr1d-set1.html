An array is a collection of elements of the same data type stored at contiguous memory locations. A 1D array in C is a linear collection where elements are accessed using a single index (e.g., array[0], array[1]).

Answer:

Advantages: Efficient access to elements (O(1) time complexity) through indexing, used to implement other data structures like stacks and queues.

Disadvantages: Fixed size (cannot be changed after declaration), inserting or deleting elements can be inefficient (O(n) time complexity).

Passing an array to a function allows the function to access and modify the array elements. In C/C++, when an array is passed to a function, it is typically passed by reference (a pointer to the first element is passed), not by value. when you pass an array, it decays to a pointer to its first element. The function receives a pointer (the pointer itself is passed by value). So the function can modify the array elements via that pointer.

Example: void f(int a[]) is equivalent to void f(int *a).
