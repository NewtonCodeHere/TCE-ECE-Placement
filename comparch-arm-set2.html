Answer: ARM instructions are 32-bit and full-featured. Thumb instructions are 16-bit, smaller in size. Thumb reduces memory usage in small systems. ARM offers higher performance and flexibility. Modern processors can switch between both modes.

Answer: ARM has 16 general-purpose registers (R0–R15). R13 is the Stack Pointer, R14 the Link Register, and R15 the Program Counter. The Program Status Register holds flags. Some registers are banked for different modes. This ensures efficient context switching.

Answer: NVIC means Nested Vectored Interrupt Controller. It manages interrupts in Cortex-M processors. It allows priority levels and nested interrupts. This improves real-time response. NVIC makes ARM efficient for embedded systems.

Answer: PWM controls average power to devices. It switches power ON and OFF quickly. Duty cycle defines ON time ratio. More duty cycle means more power delivered. It is used in motors, lights, and audio control.

Answer: Interrupts come from external devices like timers or UART. Exceptions come from CPU errors like divide-by-zero. Both pause normal execution and run handlers. They are stored in a vector table. This helps CPU handle events correctly.

Answer: DMA transfers data between memory and peripherals. It works without CPU intervention. This makes transfers faster and reduces CPU load. The CPU can do other tasks while DMA runs. Used in high-speed data communication.

Answer: ADC converts analog signals to digital numbers. DAC converts digital values back to analog voltages. ADC is used for sensors like temperature. DAC is used in audio and motor control. They connect digital systems to the real world.

Answer: A bootloader runs when the system powers on. It initializes hardware and loads the main program. It can also update firmware remotely. Bootloaders make devices easy to maintain. They are critical for embedded updates.

Answer: SCB manages system-level ARM functions. It handles exceptions and reset operations. It supports low-power and debugging features. NVIC works closely with SCB. It is an essential block in Cortex-M cores.

Answer: Cortex-M is designed for microcontrollers. It has a simple and efficient instruction set. It uses NVIC for fast interrupt handling. It supports low-power operation for IoT devices. It balances performance with cost efficiency.
