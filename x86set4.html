Answer: Minimum mode is used when a single microprocessor is in use. All control signals like RD, WR, ALE are generated by the 8086 itself.

Maximum mode is used when multiple processors or coprocessors are used. Control signals are provided by an external bus controller (8288).

Answer: 8086 has a 6-byte prefetch queue that fetches instructions in advance while the Execution Unit executes the current instruction. This pipelining improves throughput by overlapping fetching and execution.

Answer: The Stack Segment (SS) register points to the current stack segment in memory. The stack is used to store temporary data like function return addresses, parameters, and local variables during program execution.

Answer: The Direction Flag controls the direction in string operations. If DF=0, string operations process from low to high memory addresses (auto-increment). If DF=1, they process from high to low (auto-decrement).

Answer: Segment registers divide 1MB memory into logical segments each up to 64KB. The register holds the base address of the segment shifted left by 4 bits. The processor accesses memory by combining segment register values with offset addresses to get the physical address.

Answer: 8086 supports maskable (INTR) and non-maskable (NMI) interrupts. When interrupted, it pushes the current state onto the stack and jumps to the interrupt vector address to execute the Interrupt Service Routine (ISR).
